# Master Pips Academy - Demo Landing Page

Demo profesional para presentación al cliente. Diseño moderno inspirado en Axis Marketing, optimizado para SEO y listo para convertir a WordPress.

## Vista Previa

Esta es una versión **demo estática** creada para mostrar el diseño y funcionalidad al cliente. No requiere base de datos ni backend.

## Características

### Diseño
- Colores corporativos: Burgundy (#8B1538) y Dorado (#D4AF37)
- Diseño profesional sin emojis
- Inspirado en Axis Marketing
- 100% Responsive (Desktop, Tablet, Mobile)

### Contenido
- **Hero Section**: Título impactante con estadísticas
- **Programas**: 3 planes (Fundamentos, Avanzado, Mastermind)
- **Metodología**: 4 pasos del proceso de aprendizaje
- **Resultados**: Métricas de rendimiento
- **Formulario de Contacto**: Con validación y mensaje de confirmación

### Optimización
- SEO optimizado con meta tags completos
- Transiciones suaves y ligeras
- Animaciones on-scroll
- Performance optimizado (carga rápida)
- Sin dependencias externas (excepto Google Fonts)

## Instalación y Uso

### Opción 1: Abrir localmente
1. Descargar todos los archivos
2. Abrir `index.html` en cualquier navegador moderno
3. ¡Listo!

### Opción 2: GitHub Pages
1. Crear repositorio en GitHub
2. Subir los 4 archivos
3. Ir a Settings > Pages
4. Seleccionar branch main y carpeta root
5. Tu demo estará en: `https://tu-usuario.github.io/repositorio`

### Opción 3: Netlify (Recomendado - 1 minuto)
1. Ir a [netlify.com](https://netlify.com)
2. Arrastrar la carpeta con los archivos
3. ¡Demo online en segundos!

## Personalizar

### Cambiar textos
Editar `index.html`:
- Nombre de empresa: línea 37
- Hero title: línea 55
- Contenido de secciones

### Cambiar colores
Editar `styles.css` (líneas 8-17):
```css
:root {
    --primary-color: #8B1538;
    --accent-color: #D4AF37;
}
```
